# 바인그룹 2차수 실습 기준정의서

## 1. 목적

여러 채널의 상담, 등록, 매출, 환불 데이터를 한 표에서 비교하기 위한 기준을 정합니다.

## 2. 마스터 표 필수 열

| 열 이름 | 의미 | 예시 | 확인 기준 |
|---|---|---|---|
| source_channel | 데이터 출처 채널 | 홈페이지, 광고, 소개, 블로그 | 표준값으로 통일 |
| center | 센터명 | 강남센터, 분당센터, 부산센터 | 센터명 표기 통일 |
| course | 과정명 | 초등, 중등, 입시 | 과정 분류 통일 |
| month | 집계월 | 2026-07 | YYYY-MM 형식 |
| inquiries | 상담신청수 | 37 | 숫자형 |
| enrollments | 등록수 | 12 | 숫자형 |
| refunds | 환불건수 | 1 | 숫자형 |
| revenue | 매출 | 3600000 | 원 단위 |

## 3. 계산 열 기준

| 계산 열 | 공식 | 검산 방법 |
|---|---|---|
| conversion_rate | 등록수 / 상담신청수 | 첫 3행을 손계산 후 전체 적용 |
| refund_rate | 환불건수 / 등록수 | 등록수가 0이면 빈값 처리 |
| revenue_per_enrollment | 매출 / 등록수 | 등록수가 0이면 빈값 처리 |

## 4. 전처리 규칙

| 규칙 | 설명 |
|---|---|
| 원본 보존 | 원본 파일은 수정하지 않고 복사본에서 작업합니다. |
| 표준값 통일 | 강남, 강남센터, Gangnam처럼 같은 뜻의 값을 하나로 맞춥니다. |
| 민감정보 제거 | 이름, 연락처, 계약번호, 주민번호, 상세 상담 메모는 삭제하거나 샘플값으로 바꿉니다. |
| 표본 검산 | AI가 만든 수식은 3~4행을 사람이 직접 계산해 확인합니다. |

## 5. 오늘 제출 기준

완성본이 아니어도 아래 중 하나를 제출하면 됩니다.

- 업무 질문 1개
- 열 설명표 캡처
- 계산 열 수식 1개
- 피벗 기준 1개
- 막힌 지점 한 줄
