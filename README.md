# 바인그룹 생성형 AI 강의안

실시간 온라인 AI 교육용 HTML 강의안입니다.

- 1차수 주제: 업무 문서 작성 자동화 AI
- 2차수 주제: AI 데이터 처리와 분석
- 활용 도구: ChatGPT, Gemini
- 교육 대상: 바인그룹 전사 임직원
- 교육 시간: 90분

## 구성

### 1차수

- 강사 소개
- 교안 링크 안내
- 학습목표 안내
- ChatGPT와 Gemini 역할 비교
- 업무 폴더 설계
- 문서 자동화 시연 및 실습

- 데이터 분석과 전처리 개념
